to pushing borders

This is my second template for Open Web Design. Initially I was going to use the design for my own page but I decided to open it up for all. The only bug I am aware of occurs in Firefox when the text size is changed - the two top titles will not be perfectly aligned with the menu. If you're not a fan of the default blue, it's easy to change to green or orange. I've included images for the #menu a:hover (menugreen.jpg and menuorange.jpg, respectively). You'll also need to find every instance of the color "#06f" in the style sheet and change it to #6f0 for green and #f70 for orange.
